2.2 (15 points)

Implement the following in a single command:

  1. Temporarily create a new variable called num_rings which takes a value of:
  • "low" if rings < 10
  • "high" if rings > 20, and
  • "med" otherwise

  1. Group df by this new variable and sex and compute avg_weight as the average of the whole_weight + shucked_weight + viscera_weight + shell_weight for each combination of num_rings and sex.

  2. Use the geom_tile() function to create a tile plot of num_rings vs sex with the color indicating of each tile indicating the avg_weight value.

df %>% 
  mutate(
    num_rings = case_when(
    rings < 10 ~ "low", 
    rings > 20 ~ "high", 
    TRUE ~ "med")
  ) %>% 
  group_by(num_rings, sex) %>% 
  summarise(
    avg_weight = mean(
    whole_weight + 
    shucked_weight + 
    viscera_weight + 
    shell_weight
    )
  ) %>% 
  mutate(
    num_rings = factor(
      num_rings, 
      levels=c("low", "med", "high")
      )
    ) %>%
  ggplot(.) + geom_tile(aes(x=num_rings, y=sex, fill=avg_weight))
`summarise()` has grouped output by 'num_rings'. You can override using the
`.groups` argument.
